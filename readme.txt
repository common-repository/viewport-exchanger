There is plugin switching the viewport  of PC view and Mobile view.

== Description ==

This plugin is very simple.Exchang viewport by using jQuery & jQuery Cookies.

IF you installed, your responsive website can show PC view in mobile devices.

Thanks for

Using BlackFlag's jQuery.
URI:[http://black-flag.net/jquery/20140408-5086.html](http://black-flag.net/jquery/20140408-5086.html)


== Installation ==



1. Upload `viewport-exchanger` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How much viewports can exchange?  =

Only 2 viewport now.

eg).

 mobile viewport : width=device-width, minimumscale=1.0, maximum-scale=1.0

 pc viewport : width=1200
